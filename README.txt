Notes: Developed on Eclipse IDE...

- Download/install JDK
- Download install Eclipse IDE: http://www.eclipse.org/downloads/
- Download the Java Selenium client driver: http://docs.seleniumhq.org/download/
- Launch Eclipse IDE, create project
- Select project and then select 'Build Path' and then 'Configure Build Path'
- From the libraries tab, select 'Add External JARs'
- Import all the selenium jars (previously downloaded)
- Select OK

Java file can be run within the Eclipse IDE.
